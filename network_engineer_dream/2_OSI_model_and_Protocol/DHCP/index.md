# DYNAMIC HOST CONFIGURATION PROTOCOL (DHCP)

Giao thức DHCP được sử đụng để tự cấu hình các thông tin sau để máy có thể tham gia Internet:

- Địa chỉ IP private và netmask.
- Địa chỉ IP gateway.
- Danh sách các DNS server.
